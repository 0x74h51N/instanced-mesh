### Install via Package Manager

Choose your preferred package manager to install `@three.ez/instanced-mesh`:

<Tabs syncKey="pkg">
  <TabItem label="npm">
    ```sh
    npm install @three.ez/instanced-mesh
    ```
  </TabItem>

  <TabItem label="pnpm">
    ```sh
    pnpm add @three.ez/instanced-mesh
    ```
  </TabItem>

  <TabItem label="yarn">
    ```sh
    yarn add @three.ez/instanced-mesh
    ```
  </TabItem>
</Tabs>

### Install via CDN

Include the following `importmap` in your HTML file to load `@three.ez/instanced-mesh` directly from a CDN:

```html
<script type="importmap">
{
  "imports": {
    "three": "https://cdn.jsdelivr.net/npm/three/build/three.module.js",
    "three/addons/": "https://cdn.jsdelivr.net/npm/three/examples/jsm/",
    "@three.ez/instanced-mesh": "https://cdn.jsdelivr.net/npm/@three.ez/instanced-mesh/build/index.js",
    "bvh.js": "https://cdn.jsdelivr.net/npm/bvh.js/build/index.js"
  }
}
</script>
```
